# ドロップデータチェッカー

React + TypeScript + Viteのファイルやテキストデータをドロップして情報を表示するWebアプリケーションです。

### 依存関係のインストール
```bash
npm install
```

### 開発サーバーの起動
```bash
npm run dev
```

### ビルド
```bash
npm run build
```

### プレビュー
```bash
npm run preview
```

## ファイル構成
